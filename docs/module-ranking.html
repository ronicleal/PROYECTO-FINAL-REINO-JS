(inner) agruparPorNivel(jugadores, umbralopt) → {Object.<string, Array.<Jugador>>}

Agrupa la lista de jugadores según su puntuación final para determinar su nivel. Si la puntuación es igual o superior al umbral, el jugador es 'Veterano'; de lo contrario, es 'Novato'.
Parameters:
Name Type Attributes Default Description
jugadores Array.<Jugador> Lista de todos los jugadores al final de la partida.
umbral number <optional>
 300 Puntos mínimos requeridos para ser clasificado como 'Veterano'.

(inner) batalla(jugador, enemigo) → {Object}

Simula una batalla por turnos entre un jugador y un enemigo. Aplica la fórmula literal de defensa del jugador: Vida = (Vida actual + Defensa) - Ataque Enemigo. Si el jugador gana, calcula y añade los puntos obtenidos (incluyendo el multiplicador si es un Jefe).
Parameters:
Name Type Description
jugador Jugador La instancia del Jugador participante, cuya vida y puntos serán modificados.
enemigo Enemigo La instancia del Enemigo a combatir.
